## 🛠️ Setup

### Prerequisite
- Node.js 20.x
- npm 10.x

### Install node dependencies
```npm install```

### Environment variables
Create a `.env` file in the root of the project and place the values like `sample.env`

## ✅ Run

### Development environment
```npm run start:dev```

## Health check
